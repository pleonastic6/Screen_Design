# Android Studio Prototype

Kleiner nativer Android-Prototyp fuer **Share-A-Scooter Amberg** auf Basis von **Kotlin + Jetpack Compose**.

## Ziel

- kein produktionsreifes App-Setup
- sondern ein sauberer klickbarer Studien-Prototyp in Android Studio
- mit den Kernscreens aus dem Screendesign-Repo

## Enthaltene Screens

- Home / Karte
- Scooter-Detail
- Reservierung
- Vor-Ort-Auswahl
- Aktive Fahrt
- Rueckgabe

## In Android Studio oeffnen

1. Android Studio starten
2. `Open` waehlen
3. den Ordner `prototype/android-app` oeffnen
4. SDK Sync abwarten
5. Emulator starten
6. `app` ausfuehren

## Hinweise

- In dieser Umgebung konnte ich den Build nicht lokal testen, weil `java` und `gradle` hier nicht installiert sind.
- Falls Android Studio nach einem Gradle-Wrapper fragt, einmal ueber den normalen Projektimport synchronisieren lassen.
- Wenn du willst, kann ich als Naechstes echte Navigation, bessere Map-Platzhalter oder weitere Screens aus dem Wireframe-Prototypen einbauen.
